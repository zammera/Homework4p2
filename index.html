Big Dawg's Math:
Multiplication Table Creator

The Goal of this website is help you, the user, learn multiplication. Multiplication is often reffered to as repeated addition. This is because you, the user, would be adding the first number, with respects to the second number, repeatedly. There are a couple of important terms you, the user, need to know before you, the user, can run off and start creating your own multiplication tables. The multiplicand is the number that is being multiplied by another number. That other number, the one you, the user, multiply the multiplicand by is called the multiplier. Last, the result of any multiplication problem is called the product. Now that you, the user, have been informed on multiplication, it's time to make some multiplication tables. You, the user, can enter a range of multiplicand and multipliers below in the labeled fields to create a table that will show you, the user, all of the products of the multiplicands and multipliers!
